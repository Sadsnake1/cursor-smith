# Cursor Smith

Cursor Smith is a tool that lets you customize how your cursor looks and moves in your text editor. Instead of just a standard blinking line, you can choose from various styles to make your editing experience more personal and visually interesting.

It is designed to run smoothly without causing the glitches or lag that often happen with custom cursor tools.

## Available Styles

* **Line:** A clean, classic vertical line cursor that blinks smoothly as you type.
* **Underline:** A sleek horizontal dash that sits beneath your text, perfect for a minimalist look.
* **Retro Box:** A chunky, green, terminal-style block cursor. It highlights the letter you are typing by swapping its colors, making it very easy to see.
* **Torch:** A soft, warm glow that follows your cursor, highlighting the specific text you are working on while slightly dimming the rest of the line to help you stay focused.

## Key Features

* **Smooth Motion:** The cursor stretches and leaves a "trail" when you move it quickly, which you can customize to look exactly how you want.
* **Typing Feedback:** You can set a delay so the cursor stays on a character for a split second after you type it, giving your typing a more tactile feel.
* **Clean Look:** It hides the default system cursor, so the only thing you see is your custom design.

## How to Install

To set it up manually, simply copy the `manifest.json`, `main.js`, and `styles.css` files into your editor’s configuration folder.
